This server is a personalized multiple choice quiz questions, the server approaches and evaluates the user.
2 text files are needed:
1. A questions file (questions.txt), with desirable multiple choice questions - each question should start in a new row.
   The answers should be marked and listed bellow the question (a,b,c.../ 1,2,3.../ א,ב,ג.../ ....)
2. An answers file (answers.txt), with contains the correct answers in a column.
   The answers should be matched to the marked answers system.
